## 📋 Quick Start

### View Portfolio Locally
```bash
# Clone repository
git clone https://github.com/Alekhya-eng/portfolio.git
cd portfolio

# Open in browser
open index.html  # macOS
# or
start index.html  # Windows
# or
xdg-open index.html  # Linux
```

### Deploy to GitHub Pages
1. Enable GitHub Pages in repository Settings
2. Choose "Deploy from a branch" → `main` branch → root folder
3. Your site will be live at: `https://alekhya-eng.github.io/portfolio`
